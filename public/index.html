How to prepare input code for diagrams?

      

      

        
    
          
  • PlantUML C4: one @startuml … @enduml block (optional title).
    • 
          
  • Mermaid C4: first line C4Context, C4Container, or C4Component (optional mermaid Markdown code fence).
    • 
          
  • Use straight ASCII quotes ", simple IDs (letters, numbers, underscore), and one macro per line where possible.
    • 
          
  • Core macros supported in PlantUML (subset): Person, Person_Ext, System, System_Ext, SystemDb, System_Boundary, Container, ContainerDb, Component, Rel. Mermaid accepts a wider C4 set; extra lines may be ignored if not implemented.
